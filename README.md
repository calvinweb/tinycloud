# tinycloud
## What is tinycloud
tinycloud is a lightweight personal stroage solution

## Quick start
### Start from wheel
```bash
sudo pip3 install tinycloud
sudo python3 -m tinycloud.setup -c /etc/tinycloud
sudo tinycloud -c /etc/tinycloud
```
